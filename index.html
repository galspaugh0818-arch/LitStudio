Course Overview

This course introduces students to web page design using HTML and CSS. Students learn to create structured, styled web pages and apply best practices in web development.

Course Objectives

By the end of this course, students will be able to:

  1. Design web pages using HTML and CSS.
  2. Apply layout techniques for two-column and multi-column designs.
  3. Incorporate images, links, and lists effectively.
  4. Follow best practices for accessible and semantic web content.
